Конструкторы и деструкторы

Конструктор - это специальный метод, который вызывается при создании объекта класса. Он служит для инициализации объекта начальными значениями полей. Название конструктора всегда совпадает с названием класса. В C++ может быть несколько конструкторов у одного класса, каждый из которых может принимать различное число параметров или параметры разного типа.

1

class Rectangle { private: int width, height; public: // конструкторы Rectangle () { width = 0; height = 0; } Rectangle (int,int); // методы int area() { return width*height; } }; Rectangle::Rectangle (int a, int b) { width = a; height = b; } int main () { Rectangle rect1; Rectangle rect2 (3,4); cout << "area rect1: " << rect1.area() << endl; cout << "area rect2: " << rect2.area() << endl; return 0; }

Деструктор - это метод, который вызывается при удалении объекта класса. Он служит для освобождения ресурсов, выделенных для объекта, и выполнения других действий, необходимых при уничтожении объекта.

1

class Rectangle { private: int width, height; public: void set_values (int,int); int area() { return width*height; } }; void Rectangle::set_values (int x, int y) { width = x; height = y; } int main () { Rectangle rect; rect.set_values (3,4); cout << "area: " << rect.area() << endl; return 0; }
