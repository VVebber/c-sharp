Побитовые операции

Побитовые операции - это операции, которые выполняются над двоичными числами. Они используются в программировании для манипулирования отдельными битами чисел. В C# существует несколько побитовых операторов:

  1. Побитовое И (&) - выполняет логическое И над каждой парой битов двух операндов. Результатом будет число, у которого каждый бит будет равен 1 только если соответствующие биты обоих операндов равны 1.

    2. 1

    int a = 12; // 1100 в двоичной системе int b = 25; // 11001 в двоичной системе int c = a & b; // результат: 8 (01000 в двоичной системе)

  2. Побитовое ИЛИ (|) - выполняет логическое ИЛИ над каждой парой битов двух операндов. Результатом будет число, у которого каждый бит будет равен 1 если хотя бы один из соответствующих битов обоих операндов равен 1.

    3. 1

    int a = 12; // 1100 в двоичной системе int b = 25; // 11001 в двоичной системе int c = a | b; // результат: 29 (11101 в двоичной системе)

  3. Побитовое исключающее ИЛИ (^) - выполняет исключающее ИЛИ над каждой парой битов двух операндов. Результатом будет число, у которого каждый бит будет равен 1 только если соответствующие биты операндов различны.

    4. 1

    int a = 12; // 1100 в двоичной системе int b = 25; // 11001 в двоичной системе int c = a ^ b; // результат: 21 (10101 в двоичной системе)

  4. Побитовый сдвиг вправо (>>) - сдвигает все биты числа вправо на определенное количество позиций. При этом старшие биты заполняются нулями.

    5. 1

    int a = 12; // 1100 в двоичной системе int c = ~a; // результат: -13 (0011 0100 в двоичной системе)

  5. Побитовый сдвиг влево (<<) - сдвигает все биты числа влево на определенное количество позиций. При этом младшие биты заполняются нулями.

    6. 1

    int a = 12; // 1100 в двоичной системе int c = a << 2; // результат: 48 (1100 000 в двоичной системе)

  6. Побитовое отрицание (~) - инвертирует все биты числа. Если в исходном числе бит равен 0, то в результирующем числе он будет равен 1, и наоборот.

    7. 1

    int a = 12; // 1100 в двоичной системе int c = a >> 2; // результат: 3 (11 в двоичной системе) )
