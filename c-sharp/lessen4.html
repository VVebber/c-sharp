Ввод/вывод данных (I/O) - одна из наиболее важных операций при написании программ на любом языке программирования, включая C#. I/O позволяет программе получать информацию от пользователя или из внешних источников, а также выводить результаты работы программы на экран или в файлы. В этой лекции мы рассмотрим основы ввода/вывода данных в C#.

Вывод данных

Для вывода данных на экран в C# используется метод Console.WriteLine(). Этот метод выводит переданные ему аргументы на консоль, а затем переносит курсор на новую строку. Вот простейший пример:

1

Console.WriteLine("Hello, World!");

Этот код выведет на экран строку "Hello, World!".

Метод Console.Write() похож на Console.WriteLine(), но не переносит курсор на новую строку. Вот пример:

1

Console.Write("Enter your name: ");

Этот код выведет на экран строку "Enter your name: " и остановится, ожидая ввода данных.

Если вам нужно вывести данные в формате, отличном от строки, можно использовать метод Console.WriteFormat(). Этот метод работает так же, как и Console.WriteLine(), но принимает дополнительный аргумент - строку формата. Вот пример:

1

string name = "John"; int age = 30; Console.WriteFormat("My name is {0} and I'm {1} years old.", name, age);

Этот код выведет на экран строку "My name is John and I'm 30 years old.".

Ввод данных

Для ввода данных с консоли в C# используется метод Console.ReadLine(). Этот метод ожидает ввода данных от пользователя, затем считывает введенную строку и возвращает ее как результат. Вот пример:

1

Console.Write("Enter your name: "); string name = Console.ReadLine(); Console.WriteLine("Hello, {0}!", name);

Этот код запросит у пользователя его имя, затем выведет на экран строку "Hello, {имя}!".

Если вам нужно считать значение, отличное от строки, вы можете воспользоваться методом Parse() для преобразования введенной строки в нужный тип данных. Например:

1

Console.Write("Enter your age: "); int age = int.Parse(Console.ReadLine()); Console.WriteLine("You are {0} years old.", age);

Этот код запросит у пользователя его возраст, затем выведет на экран строку "You are {возраст} years old.".

Файловый ввод/вывод

В C# также есть возможность работать с файлами для ввода/вывода данных. Для работы с файлами необходимо использовать классы System.IO.File и System.IO.StreamReader/StreamWriter. Вот примеры:

Чтение данных из файла

1

// открываем файл для чтения string fileName = "input.txt"; StreamReader reader = new StreamReader(fileName); // считываем данные из файла string line; while ((line = reader.ReadLine()) != null) { Console.WriteLine(line); } // закрываем файл reader.Close();

Этот код открывает файл "input.txt" для чтения, считывает данные построчно и выводит их на экран. После окончания чтения файл закрывается.

Для записи данных в файл используется класс StreamWriter. Вот пример:

1

// открываем файл для записи string fileName = "output.txt"; StreamWriter writer = new StreamWriter(fileName); // записываем данные в файл string message = "Hello, World!"; writer.WriteLine(message); // закрываем файл writer.Close();

Этот код открывает файл "output.txt" для записи, записывает в него строку "Hello, World!" и закрывает файл.

Обработка ошибок ввода/вывода

При работе с вводом/выводом данных могут возникать ошибки, такие как отсутствие файла или некорректный формат данных. Для обработки ошибок можно использовать блок try-catch. Вот пример:

1

try { // открываем файл для чтения string fileName = "input.txt"; StreamReader reader = new StreamReader(fileName); // считываем данные из файла string line; while ((line = reader.ReadLine()) != null) { Console.WriteLine(line); } // закрываем файл reader.Close(); } catch (Exception ex) { Console.WriteLine("An error occurred: {0}", ex.Message); }

Этот код открывает файл "input.txt" для чтения и считывает данные из него. Если при этом происходит ошибка, она перехватывается блоком catch, и на экран выводится сообщение об ошибке.
