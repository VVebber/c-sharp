Константы и readonly поля

Константы и readonly поля - это способы определения переменных в C#, которые не могут быть изменены во время выполнения программы. Однако, есть некоторые различия между ними.

Константа

Константы - это переменные, которые задаются единожды во время компиляции программы и не могут быть изменены во время выполнения программы. Они объявляются с помощью ключевого слова const, за которым следует тип данных, имя и значение константы. Пример:

C#

const double Pi = 3.14159;
const Pi=31,41; // Ошика, нельзя изменять.

Поля readonly

Поля только для чтения (readonly) - это переменные, значения которых могут быть установлены только во время выполнения конструктора и не могут быть изменены в дальнейшем. Они могут быть определены только в теле класса, но не в методах. Для создания поля только для чтения в C# используется ключевое слово readonly. Пример:

C#

class MyClass { readonly int myReadOnlyField; public MyClass(int value) { myReadOnlyField = value; } }
