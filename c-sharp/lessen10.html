Тернальные операции

Тернарные операции в языке C# позволяют выполнять условные вычисления на основе результатов сравнения или логических операций. Тернарная операция имеет следующий синтаксис:

1

condition ? value1 : value2

Если условие condition истинно, тогда возвращается значение value1, в противном случае возвращается значение value2. Давайте рассмотрим несколько примеров использования тернарной операции в C#.

Пример 1:

1

int age = 17; string message = (age >= 18) ? "Вы совершеннолетний" : "Вы несовершеннолетний"; Console.WriteLine(message); // Выводит "Вы несовершеннолетний"

В этом примере мы используем тернарную операцию для проверки возраста. Если возраст больше или равен 18, то мы выводим сообщение "Вы совершеннолетний", в противном случае мы выводим сообщение "Вы несовершеннолетний".

Пример 2:

1

int number = 7; string result = (number % 2 == 0) ? "четное" : "нечетное"; Console.WriteLine($"Число {number} - {result}"); // Выводит "Число 7 - нечетное"

В этом примере мы используем тернарную операцию для проверки четности числа. Если число number делится на 2 без остатка, то мы выводим сообщение "четное", в противном случае мы выводим сообщение "нечетное".

Пример 3:

1

string input = "OpenAI"; string result = (input.Length > 10) ? "Длинное слово" : "Короткое слово"; Console.WriteLine(result); // Выводит "Короткое слово"

В этом примере мы используем тернарную операцию для проверки длины строки input. Если длина строки больше 10 символов, то мы выводим сообщение "Длинное слово", в противном случае мы выводим сообщение "Короткое слово".

Тернарные операции в C# могут быть очень полезными при написании условных выражений в одну строку кода. Однако, не следует злоупотреблять этим приемом, так как слишком сложные тернарные операции могут затруднить чтение и понимание кода другими программистами.
