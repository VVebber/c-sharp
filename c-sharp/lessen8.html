Логические операции

В C# логические операции используются для проверки и управления логическими выражениями, которые могут принимать только два значения: true (истина) и false (ложь).

  1. && (логическое И): Возвращает true, если оба операнда истинны.
  2. || (логическое ИЛИ): Возвращает true, если хотя бы один из операндов истинен.
  3. ! (логическое НЕ): Инвертирует значение операнда (если операнд true, то результат будет false и наоборот).
