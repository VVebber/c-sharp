Массивы

Одномерные массивы

Одномерный массив - это структура данных, которая содержит элементы одного типа и располагается в памяти непрерывно. Для доступа к элементам одномерного массива используется индексация по одному индексу.

Создание одномерного массива

Одномерный массив можно создать с помощью оператора `new`, указав тип элементов массива и его размер:

1

int[] array = new int[5];

Этот код создает массив целых чисел `int` размера 5. После создания массива все его элементы инициализируются значением по умолчанию для типа элементов (для `int` это `0`).

Также можно создать одномерный массив с помощью инициализатора массива:

1

int[] array = new int[] { 1, 2, 3, 4, 5 };

Или сокращенной формой:

1

int[] array = { 1, 2, 3, 4, 5 };

Доступ к элементам одномерного массива

Доступ к элементам одномерного массива осуществляется с помощью индексации. Индексация начинается с нуля: Доступ к элементам одномерного массива осуществляется с помощью индексации. Индексация начинается с нуля:

1

int[] array = { 1, 2, 3, 4, 5 }; int firstElement = array[0]; // получаем первый элемент массива, равный 1 int lastElement = array[4]; // получаем последний элемент массива, равный 5

1.3 Изменение элементов одномерного массива

Элементы одномерного массива можно изменять, присваивая новые значения по индексу:

1

int[] array = { 1, 2, 3, 4, 5 }; array[2] = 10; // изменяем третий элемент массива, он становится равным 10

Примеры использования одномерных массивов

Одномерные массивы могут быть использованы в различных задачах, где требуется хранение последовательности значений. Например, можно использовать массив для хранения оценок студентов:

1

int[] grades = new int[5]; grades[0] = 5; grades[1] = 4; grades[2] = 3; grades[3] = 5; grades[4] = 2;

Многомерные массивы

Многомерный массив - это структура данных, которая представляет собой таблицу элементов, упорядоченных по нескольким измерениям. В C# многомерные массивы могут иметь любое количество измерений.

Создание многомерного массива

Многомерный массив можно создать с помощью оператора `new`, указав тип элементов массива, количество измерений и размеры каждого измерения:

1

int[,] matrix = new int[3, 4];

Этот код создает двумерный массив `int` размером 3 на 4.

Многомерный массив также можно создать с помощью инициализатора массива. Для этого нужно указать размерность массива и инициализировать его элементы:

1

int[,] matrix = { { 1, 2, 3 }, { 4, 5, 6 } };

Этот код создает двумерный массив размером 2 на 3 и заполняет его значениями от 1 до 6.

Доступ к элементам многомерного массива

Доступ к элементам многомерного массива осуществляется с помощью индексации по каждому измерению:

1

int[,] matrix = { { 1, 2, 3 }, { 4, 5, 6 } }; int element = matrix[1, 2]; // получаем элемент массива, равный 6

Изменение элементов многомерного массива

Элементы многомерного массива можно изменять, присваивая новые значения по индексу:

1

int[,] matrix = { { 1, 2, 3 }, { 4, 5, 6 } }; matrix[1, 2] = 10; // изменяем элемент массива, он становится равным 10

Примеры использования многомерных массивов

Многомерные массивы могут быть использованы для хранения таблиц и матриц. Например, можно использовать массив для хранения матрицы смежности графа:

1

int[,] graph = new int[5, 5]; graph[0, 1] = 1; graph[1, 2] = 1; graph[2, 3] = 1; graph[3, 4] = 1;

Зубчатые массивы

Зубчатый массив - это массив массивов, где каждый подмассив может иметь разный размер. В C# зубчатые массивы создаются с помощью оператора `new` и инициализируются как одномерный массив ссылок на другие одномерные массивы:

1

int[][] jaggedArray = new int[3][]; jaggedArray[0] = new int[2] { 1, 2 }; jaggedArray[1] = new int[3] { 3, 4, 5 }; jaggedArray[2] = new int[4] { 6, 7, 8, 9 };

Этот код создает зубчатый массив, состоящий из трех одномерных массивов с разным количеством элементов.

Доступ к элементам зубчатого массива

1

int[][] jaggedArray = new int[3][]; jaggedArray[0] = new int[2] { 1, 2 }; jaggedArray[1] = new int[3] { 3, 4, 5 }; jaggedArray[2] = new int[4] { 6, 7, 8, 9 }; int element = jaggedArray[1][2]; // получаем элемент массива, равный 5

Изменение элементов зубчатого массива

Элементы зубчатого массива можно изменять, присваивая новые значения по индексу:

1

int[][] jaggedArray = new int[3][]; jaggedArray[0] = new int[2] { 1, 2 }; jaggedArray[1] = new int[3] { 3, 4, 5 }; jaggedArray[2] = new int[4] { 6, 7, 8, 9 }; jaggedArray[1][2] = 10; // изменяем элемент массива, он становится равным 10

Примеры использования зубчатых массивов

Зубчатые массивы могут быть использованы для хранения набора данных, которые имеют разный размер. Например, можно использовать зубчатый массив для хранения информации о клиентах в интернет-магазине:

1

string[][] customers = new string[3][]; customers[0] = new string[2] { "John Doe", "johndoe@example.com" }; customers[1] = new string[3] { "Jane Smith", "janesmith@example.com", "555-1234" }; customers[2] = new string[4] { "Bob Johnson", "bjohnson@example.com", "555-4321", "New York" };

В этом примере каждый элемент массива содержит информацию о клиенте, которая может иметь разное количество полей.

Создание неограниченных массивов с помощью `List`

Коллекция `List` может использоваться для хранения неограниченного количества элементов любого типа данных `T`. При создании объекта `List` не нужно задавать размер, и можно добавлять новые элементы динамически:

1

List numbers = new List(); numbers.Add(1); // добавляем элемент в список numbers.Add(2);

Доступ к элементам неограниченных массивов

Доступ к элементам `List` осуществляется также, как и к элементам массива, с помощью индексации:

1

List numbers = new List(); numbers.Add(1); numbers.Add(2); int secondElement = numbers[1]; // получаем второй элемент списка, который равен 2

Изменение элементов неограниченных массивов

Элементы `List` также можно изменять, присваивая новые значения по индексу:

1

List numbers = new List(); numbers.Add(1); numbers.Add(2); int secondElement = numbers[1]; // получаем второй элемент списка, который равен 2

Примеры использования неограниченных массивов

`List ` может быть использован для хранения любых типов данных, и он удобен в тех случаях, когда нужно добавлять или удалять элементы динамически. Например, можно использовать `List` для хранения списка имен пользователей:

1

List names = new List(); names.Add("John"); names.Add("Jane"); names.Add("Bob");
