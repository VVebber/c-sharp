Массивы

Одномерные массивы

Одномерный массив - это структура данных, которая содержит элементы одного типа и располагается в памяти непрерывно. Для доступа к элементам одномерного массива используется индексация по одному индексу.

Создание одномерного массива

Одномерный массив можно создать с помощью оператора `new`, указав тип элементов массива и его размер:

1

int[] array = new int[5];

Этот код создает массив целых чисел `int` размера 5. После создания массива все его элементы инициализируются значением по умолчанию для типа элементов (для `int` это `0`).

Также можно создать одномерный массив с помощью инициализатора массива:

1

int[] array = new int[] { 1, 2, 3, 4, 5 };

Или сокращенной формой:

1

int[] array = { 1, 2, 3, 4, 5 };
