Целочисленные типы данных

В языке C# целочисленные типы данных представлены следующими ключевыми словами:

  • sbyte - знаковый 8-битный целочисленный тип данных. Может принимать значения от -128 до 127.
  • byte - беззнаковый 8-битный целочисленный тип данных. Может принимать значения от 0 до 255.
  • short - знаковый 16-битный целочисленный тип данных. Может принимать значения от -32768 до 32767.
  • ushort - беззнаковый 16-битный целочисленный тип данных. Может принимать значения от 0 до 65535.
  • int - знаковый 32-битный целочисленный тип данных. Может принимать значения от -2147483648 до 2147483647.
  • uint - беззнаковый 32-битный целочисленный тип данных. Может принимать значения от 0 до 4294967295.
  • long - знаковый 64-битный целочисленный тип данных. Может принимать значения от -9223372036854775808 до 9223372036854775807.
  • ulong - беззнаковый 64-битный целочисленный тип данных. Может принимать значения от 0 до 18446744073709551615.

int integerNumber = 42;

Каждый целочисленный тип данных в C# занимает определенное количество памяти в зависимости от его размера. Например, byte занимает 1 байт (8 бит), а long занимает 8 байтов (64 бита).
Также в языке C# есть модификаторы signed и unsigned, которые можно использовать вместо ключевых слов sbyte, short, int и long. Например, можно определить знаковый 32-битный целочисленный тип данных с помощью ключевых слов int или signed int.

Вещественные типы данных

В языке C# существуют три вещественных типа данных:

  • float - представляет 64-битное число с плавающей точкой. Он может хранить числа от -1.7 × 10^308 до 1.7 × 10^308 и имеет точность около 15 знаков после запятой.
  • double - представляет 64-битное число с плавающей точкой. Он может хранить числа от -1.7 × 10^308 до 1.7 × 10^308 и имеет точность около 15 знаков после запятой.
  • decimal - представляет 128-битное число с плавающей точкой. Он может хранить числа от ±1.0 × 10^-28 до ±7.9 × 10^28 и имеет точность около 28 знаков после запятой.

Вещественные типы данных используются для представления чисел с плавающей точкой. При работе с вещественными типами данных необходимо учитывать особенности округления и погрешности вычислений, связанные с использованием чисел с плавающей точкой.

Типы данных для символов и строк

Оба типа данных имеют важное значение при работе с текстовыми данными в C#.

  • char представляет одиночный символ в формате Unicode, который занимает 2 байта в памяти. Он может быть инициализирован как символом, так и его числовым значением в десятичной или шестнадцатеричной системе счисления.
  • string представляет набор символов Unicode, которые образуют строку. Он может быть инициализирован с помощью двойных кавычек и поддерживает множество методов для работы со строками.

char myChar = 'A';

string message = "Hello, world!";

Логический тип данных

Логический тип данных (bool) в C# может принимать только два значения: true (истина) и false (ложь). Он используется для представления логических выражений, результатом которых может быть только одно из двух значений - истина или ложь.

C#

1

2

bool isTrue = true;

bool isFalse = false;

Логические типы данных используются в условных операторах (if, while, for, switch) для проверки условий и принятия решений на основе этих условий. Также они используются в булевых выражениях для объединения и инвертирования условий.
