Циклы

Циклы – это один из наиболее часто используемых элементов языка программирования C#, который позволяет выполнять повторяющиеся операции несколько раз.

C# поддерживает три вида циклов: `for`, `while` и `do-while`. Каждый из них имеет свои особенности и может использоваться в различных ситуациях.

Цикл for

Цикл `for` используется для повторения блока кода заданное количество раз. Синтаксис цикла for выглядит следующим образом:

1

for (инициализация; условие; итерация) { // тело цикла }

  • `инициализация` - выполняется только один раз перед началом цикла и используется для инициализации переменных.
  • `условие` - определяет, должен ли цикл продолжаться. Если условие истинно, то тело цикла выполняется, иначе цикл завершается.
  • `итерация` - выполняется после каждого выполнения тела цикла и используется для изменения переменных.

Пример использования цикла `for`:

1

for (int i = 1; i <= 10; i++) { Console.WriteLine(i); }

В этом примере мы создаем переменную `i` и инициализируем ее значением 1. Условие цикла указывает, что цикл должен выполняться, пока значение переменной `i` меньше или равно 10. После каждого выполнения тела цикла значение переменной `i` увеличивается на 1. В результате на консоль будут выведены числа от 1 до 10.

Цикл while

Цикл `while` используется для повторения блока кода до тех пор, пока определенное условие истинно. Синтаксис цикла while выглядит следующим образом:

1

while (условие) { // тело цикла }

  • `условие` – определяет, должен ли цикл продолжаться. Если условие истинно, то тело цикла выполняется, иначе цикл завершается.

1

int i = 1; while (i <= 10) { Console.WriteLine(i); i++; }

В этом примере мы создаем переменную `i` и инициализируем ее значением 1. Условие цикла указывает, что цикл должен выполняться, пока значение переменной `i` меньше или равно 10. После каждого выполнения тела цикла значение переменной `i` увеличивается на 1. В результате на консоль будут выведены числа от 1 до 10.

Цикл do-while

1

do { // тело цикла } while (условие);

  • `условие` – определяет, должен ли цикл продолжаться. Если условие истинно, то тело цикла выполняется, иначе цикл завершается.

Пример использования цикла `do-while`:

1

int i = 1; do { Console.WriteLine(i); i++; } while (i <= 10);

В этом примере мы создаем переменную `i` и инициализируем ее значением 1. Тело цикла выполняется хотя бы один раз, так как условие не проверяется до первого выполнения тела цикла. Условие цикла указывает, что цикл должен выполняться, пока значение переменной `i` меньше или равно 10. После каждого выполнения тела цикла значение переменной `i` увеличивается на 1. В результате на консоль будут выведены числа от 1 до 10.

Прерывание цикла

Иногда возникает необходимость прервать выполнение цикла до завершения всех итераций. Для этого в C# используется оператор break. Он позволяет выйти из цикла, даже если условие еще не стало ложным. Пример использования оператора `break`:

1

for (int i = 1; i <= 10; i++) { if (i == 5) { break; } Console.WriteLine(i); }

В этом примере цикл `for` выполняется до тех пор, пока значение переменной `i` меньше или равно 10. Если значение переменной `i` равно 5, то оператор `break` прерывает выполнение цикла. В результате на консоль будут выведены числа от 1 до 4.

Еще один оператор, который может использоваться в циклах – это `continue`. Он позволяет пропустить текущую итерацию цикла и перейти к следующей. Пример использования оператора `continue`:

1

for (int i = 1; i <= 10; i++) { if (i % 2 == 0) { continue; } Console.WriteLine(i); }

В этом примере цикл `for` выполняется до тех пор, пока значение переменной `i` меньше или равно 10. Если значение переменной `i` является четным, то оператор continue пропускает текущую итерацию цикла, и переходит к следующей. В результате на консоль будут выведены нечетные числа от 1 до 9.

Циклы foreach

Цикл `foreach` используется для перебора элементов коллекций или массивов. Синтаксис цикла `foreach` выглядит следующим образом:

1

foreach (тип_элемента переменная_элемента in коллекция) { // тело цикла }

  • `тип_элемента` – тип элементов в коллекции;
  • `переменная_элемента` – переменная, которая будет хранить текущий элемент коллекции;
  • `коллекция` – коллекция или массив, который будет перебираться в цикле.

Пример использования цикла `foreach` для перебора элементов массива:

1

int[] numbers = { 1, 2, 3, 4, 5 }; foreach (int number in numbers) { Console.WriteLine(number); }

В этом примере мы создаем массив numbers, который содержит пять целочисленных элементов. Затем мы используем цикл `foreach`, чтобы перебрать элементы массива и вывести их на консоль.

Пример использования цикла `foreach` для перебора элементов списка:

1

List names = new List() { "Alice", "Bob", "Charlie" }; foreach (string name in names) { Console.WriteLine(name); }

В этом примере мы создаем список `names`, который содержит три строки. Затем мы используем цикл `foreach`, чтобы перебрать элементы списка и вывести их на консоль.

Использование циклов является важной частью программирования, и хорошее знание работы с циклами может существенно улучшить производительность и качество вашего кода.
