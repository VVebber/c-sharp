Консольный вывод

Консольный вывод в языке C# позволяет отображать текстовую информацию в консольном окне. Это может быть полезно для отладки приложений или для создания текстового интерфейса для пользователей.

Для вывода текста на консоль используется метод Console.WriteLine(). Он позволяет вывести текст на новой строке и автоматически перейти на следующую. Если же нужно вывести текст без перевода на новую строку, то используется метод Console.Write().

Примеры использования методов:

C#

// Вывод текста на консоль с переходом на новую строку
Console.WriteLine("Hello, world!");

// Вывод текста на консоль без перехода на новую строку
Console.Write("Enter your name: ");

Кроме того, методы Console.WriteLine() и Console.Write() могут принимать аргументы, которые будут автоматически преобразованы в строку. Например:

C#

int age = 25;
Console.WriteLine("My age is: " + age);

double temperature = 36.6;
Console.WriteLine("Body temperature: {0} C", temperature);

В первом примере переменная age автоматически преобразуется в строку и выводится на консоль. Во втором примере используется форматированный вывод, где {0} заменяется на значение переменной temperature.

Ввод даных

Консольный ввод - это способ ввода данных в программу через командную строку или терминал. В C# консольный ввод можно реализовать с помощью класса Console.

Для чтения данных из консоли в C# используется метод Console.ReadLine(). Этот метод считывает следующую строку из входного потока консоли и возвращает ее в качестве строки.

Например, следующий код читает строку, введенную пользователем, и сохраняет ее в переменной input:

C#

string input = Console.ReadLine();

Важно отметить, что метод Console.ReadLine() всегда возвращает строку, даже если пользователь ввел число или другой тип данных. Если требуется работать с другими типами данных, необходимо преобразовать введенную строку в нужный тип, используя соответствующие методы преобразования типов данных, такие как:

  • ToInt32: Преобразует значение указанного string в эквивалентное 32-разрядное целое число со знаком.
  • ToDouble: Преобразует значение указанного string к эквивалентному числу с плавающей запятой двойной точности.
  • ToBoolean: Преобразует значение указанного string в эквивалентное логическое значение.
  • ToString: Преобразует значение указанного объекта в его эквивалентное строковое представление.
C#

Console.Write("Введите имя: ");
string? name = Console.ReadLine();

Console.Write("Введите возраст: ");
int age = Convert.ToInt32(Console.ReadLine());

Console.Write("Введите рост: ");
double height = Convert.ToDouble(Console.ReadLine());

Console.Write("Введите размер зарплаты: ");
decimal salary = Convert.ToDecimal(Console.ReadLine());
