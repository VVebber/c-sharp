Конструкции if/else

Конструкция if/else является одним из фундаментальных элементов языка C#. Она позволяет выполнить определенный блок кода, только если определенное условие истинно. Если условие ложно, то будет выполнен другой блок кода.

Синтаксис конструкции if/else выглядит следующим образом:

1

if (условие) { // Блок кода, который будет выполнен, если условие истинно } else { // Блок кода, который будет выполнен, если условие ложно }

Здесь условие - это выражение, которое должно вычисляться в логическое значение (true/false). Если выражение истинно, то будет выполнен первый блок кода (между фигурными скобками после if). Если выражение ложно, то будет выполнен второй блок кода (между фигурными скобками после else).

Например, давайте напишем программу, которая проверяет, является ли число четным или нечетным, и выводит соответствующее сообщение на экран:

1

int number = 7; if (number % 2 == 0) { Console.WriteLine("Число " + number + " четное"); } else { Console.WriteLine("Число " + number + " нечетное"); }

Здесь мы используем оператор % для проверки, делится ли число на 2 без остатка. Если да, то это значит, что число четное. Если нет, то оно нечетное. В зависимости от результата проверки, будет выполнен соответствующий блок кода.

Иногда может возникнуть ситуация, когда нужно выполнить несколько проверок условий. Для этого можно использовать вложенные конструкции if/else:

1

int age = 25; if (age < 18) { Console.WriteLine("Вы еще не достигли совершеннолетия"); } else { if (age < 25) { Console.WriteLine("Вы молоды и полны сил"); } else { Console.WriteLine("Вы уже взрослый человек"); } }

Здесь мы проверяем возраст и выводим соответствующее сообщение на экран, в зависимости от того, находится ли возраст в определенном диапазоне. Если возраст меньше 18, то выводится сообщение о том, что человек еще не достиг совершеннолетия. Если возраст больше или равен 18, то мы переходим к следующей проверке, где проверяем, находится ли возраст в диапазоне от 18 до 25. Если да, то выводим сообщение о том, что человек молод и полон сил. И если возраст больше или равен 25, то выводится сообщение о том, что человек уже взрослый.

Кроме того, конструкцию if/else можно использовать внутри других конструкций if/else, чтобы создавать более сложную логику выполнения кода:

1

int a = 5; int b = 10; if (a < b) { if (a < 0) { Console.WriteLine("Число a отрицательное"); } else { Console.WriteLine("Число a положительное"); } } else { if (b < 0) { Console.WriteLine("Число b отрицательное"); } else { Console.WriteLine("Число b положительное"); } }

Здесь мы сравниваем два числа a и b. Если a меньше b, то мы проверяем, является ли a отрицательным или положительным числом. Если a отрицательное, то выводим сообщение об этом. Если нет, то выводим сообщение о том, что a положительное. Если a не меньше b, то мы проверяем, является ли b отрицательным или положительным числом, и выводим соответствующее сообщение.

Также существует конструкция if/else if, которая позволяет проверять несколько условий последовательно:

1

int grade = 4; if (grade == 5) { Console.WriteLine("Отлично!"); } else if (grade == 4) { Console.WriteLine("Хорошо"); } else if (grade == 3) { Console.WriteLine("Удовлетворительно"); } else if (grade == 2) { Console.WriteLine("Неудовлетворительно"); } else { Console.WriteLine("Оценка некорректна"); }

Здесь мы проверяем оценку grade и выводим соответствующее сообщение на экран, в зависимости от того, какая оценка была получена. Если оценка равна 5, то выводится сообщение "Отлично!". Если оценка равна 4, то выводится сообщение "Хорошо". Если оценка равна 3, то выводится сообщение "Удовлетворительно". Если оценка равна 2, то выводится сообщение "Неудовлетворительно". Если же ни одно из условий не выполняется, то выводится сообщение о том, что оценка некорректна.

Конструкция if/else очень важна для написания условных выражений в C#. Она позволяет написать логику, которая будет выполняться только в том случае, если определенное условие истинно. Благодаря этому мы можем контролировать выполнение программы и обрабатывать ошибки и некорректные входные данные.

Таким образом, конструкция if/else является одним из основных инструментов, которые позволяют контролировать выполнение программы в зависимости от определенных условий. Она позволяет создавать логику, которая будет выполняться только в том случае, если определенное условие истинно
